标签外挂（Tag Plugins）

标签外挂是Hexo独有的功能，并不是标准的Markdown格式。
以下的写法，只适用于Butterfly主题，用在其它主题上不会有效果，甚至可能会报错。 使用前请留意

Tabs

名称 用法
Unique name 选项卡的唯一名称,不允许符号
index 如果未指定，将选择第一个选项卡；如果索引为-1，则不会选择任何选项卡
Tab caption 选项卡的标题，如果未指定标题，则自动编号。如果未指定标题，而是指定了图标，则为空
@icon FontAwesome图标名称

tab名字为第一个Tab

只有图标 沒有Tab名字

名字+icon

Note

修改 主題配置文件

名称 用法
class 【可选】标识，不同的标识有不同的配色（ default / primary / success / info / warning / danger ）
no-icon 【可选】不显示 icon
style 【可选】可以覆盖配置中的 style（simple/modern/flat/disabled）

默认 提示块标签

default 提示块标签

primary 提示块标签

success 提示块标签

info 提示块标签

warning 提示块标签

danger 提示块标签

名称 用法
color 【可选】顔色(default / blue / pink / red / purple / orange / green)
icon 【可选】可配置自定义 icon (只支持 fontawesome 图标, 也可以配置 no-icon )
style 【可选】可以覆盖配置中的 style（simple/modern/flat/disabled）

Butterfly安裝方法

在你的博客根目錄里

git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly

如果想要安裝比較新的dev分支，可以

git clone -b dev https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly

Button

名称 用法
url 链接
text 按钮文字
icon 【可选】图标
color 【可选】按钮背景顔色(style）;按钮字體和邊框顔色(outline);default/blue/pink/red/purple/orange/green
style 【可选】按钮样式;outline/留空
layout 【可选】按钮布局;block/留空
position 【可选】按钮位置;center/right/留空
size 【可选】按钮大小;larger/留空
